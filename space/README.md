# Symptom Triage Coach

A LoRA fine-tune of Qwen2.5-1.5B that takes a plain-language symptom description and returns a structured pre-visit prep response: body systems to consider, possible causes, red flags, and questions the doctor will likely ask.

**Not medical advice.** This is a research/portfolio demo, not a diagnostic tool. For any real medical concern, see a licensed clinician.

Source code, training data, and evaluation: [github.com/ksolano220/symptom-triage-coach](https://github.com/ksolano220/symptom-triage-coach)
